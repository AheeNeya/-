### Типографика:
```
Заголовки: 
  - Шрифт: "Orbitron" или "Rajdhani" или "Exo 2" (футуристичный)
  - С эффектом неонового свечения (text-shadow)

  - Шрифт: "Inter" или "JetBrains Mono" (для кодерского вайба)

  - Моноширинный для "хакерских" элементов
  - JetBrains Mono / Fira Code

### Эффекты (ОБЯЗАТЕЛЬНЫЕ):
```
1. Glassmorphism:
   - background: rgba(255,255,255,0.03-0.08)
   - backdrop-filter: blur(10-20px)
   - border: 1px solid rgba(255,255,255,0.1)

2. Неоновое свечение:
   - box-shadow: 0 0 10px #00f0ff, 0 0 20px #00f0ff44
   - text-shadow: 0 0 10px #00f0ff, 0 0 20px #00f0ff66
   - Пульсация (animation: pulse)

3. Голографические переливы:
   - background: linear-gradient с анимацией
   - Перелив цветов при движении мыши (mousemove parallax)
   - Радужный отблеск на картах

4. 3D эффекты:
   - perspective + rotateX/Y на карточках
   - Tilt-эффект при наведении мыши (3D поворот к курсору)
   - Parallax слои (фон двигается медленнее)
   - Transform3d для глубины

5. Частицы:
   - Лёгкие плавающие частицы на фоне
   - Или цифровой "дождь" из символов (лёгкий, не мешающий)
   - Или звёздное небо с мерцанием

6. Анимации появления:
   - Элементы появляются при скролле (Intersection Observer)
   - Fade in + slide up / scale up
   - Stagger эффект (элементы появляются по очереди)

7. Hover эффекты:
   - Карточки: 3D tilt + усиление свечения
   - Кнопки: неоновый pulse + scale
   - Ссылки: подчёркивание неоновой линией

8. Курсор:
   - Кастомный курсор (неоновая точка + trail)
   - Или свечение вокруг курсора на тёмном фоне

9. Глитч-эффект:
   - На заголовке имени (периодический)
   - При наведении на определённые элементы
   - Не постоянный — а как "цифровая помеха"
```
